KADEME 6 - YÖNETİCİ DASHBOARD

Eklenenler:
- Sadece yönetici kullanımına yönelik modern dashboard iskeleti eklendi.
- Sol sidebar ve gerçek Font Awesome ikonları eklendi.
- Genel Bakış, Lisanslar, Cihaz Kilidi, Güncellemeler, Ayarlar menüleri hazırlandı.
- Toplam hesap, aktif lisans, yakında bitecek lisans ve bağlı cihaz kartları eklendi.
- Dashboard istatistikleri SQLite veritabanından okunacak şekilde bağlandı.
- Çıkış Yap butonu eklendi.

Notlar:
- Login tasarımına ve mevcut giriş/kayıt/şifre değiştirme akışına dokunulmadı.
- Bu aşama müşteri paneli değildir; sadece yönetici ekranıdır.
- Lisans listeleme, lisans oluşturma ekranı ve HWID reset ekranı sonraki kademede detaylandırılabilir.

Çalıştırma:
npm install
npm start

EXE üretme:
npm run dist
