Kademe 7.2 - Aktivasyon Logları

Eklenenler:
- activation_logs SQLite tablosu.
- Başarılı admin girişi log kaydı.
- Hatalı giriş log kaydı.
- Banlı/pasif kullanıcı giriş denemesi log kaydı.
- Lisans süresi dolmuş/aktif lisans yok giriş denemesi log kaydı.
- HWID uyuşmazlığı log kaydı.
- HWID reset log kaydı.
- Lisans oluşturma/uzatma log kaydı.
- Kullanıcı banlama/aktif etme/silme log kaydı.
- Admin panelde Aktivasyon Logları tablosu.

Not:
- Login, müşteri paneli ve mevcut yönetim akışı değiştirilmedi.
